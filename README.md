# 👁 all-seeing Telegram Bot

Бот работает в **Business Mode** (Telegram Premium). Отслеживает отредактированные и удалённые сообщения в личных чатах владельца бота, а также фиксирует новых пользователей с их Telegram ID.

---

## 🌟 Возможности

- **Отслеживание отредактированных сообщений** — показывает было/стало
- **Отслеживание удалённых сообщений** — сохраняет оригинал
- **Определение новых пользователей** — присылает имя и ID
- **Поддержка нескольких языков** — `en`, `ru`

---

## ⚙️ Установка

### 1. Создайте бота и включите Business Mode

1. Напишите [@BotFather](https://t.me/botfather) → `/newbot`
2. Скопируйте токен
3. Перейдите: `Bot Settings → Business Mode → Turn on`
4. В Telegram: `Настройки → Telegram Business → Чат-боты` → добавьте своего бота

### 2. Узнайте свой Telegram ID

Напишите [@userinfobot](https://t.me/userinfobot) — он пришлёт ваш числовой ID.

### 3. Клонируйте репозиторий

```bash
git clone https://github.com/ВАШ_USERNAME/ВАШ_РЕПОЗИТОРИЙ
cd ВАШ_РЕПОЗИТОРИЙ
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Настройте переменные окружения

Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

Откройте `.env` и заполните:

```env
BOT_TOKEN=токен_от_botfather        # обязательно
ADMIN_ID=ваш_числовой_id            # обязательно
TIMEZONE=Europe/Moscow              # необязательно, по умолчанию UTC
LANGUAGE=ru                         # необязательно, по умолчанию en
```

> ⚠️ Файл `.env` добавлен в `.gitignore` — он никогда не попадёт в репозиторий.

### 6. Запустите бота

```bash
python main.py
```

---

## 🌐 Добавление нового языка

1. Скопируйте `languages/en.py` → `languages/XX.py`
2. Переведите строки
3. Установите `LANGUAGE=XX` в `.env`
